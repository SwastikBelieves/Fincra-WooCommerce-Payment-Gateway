# Fincra WooCommerce Payment Gateway

[![License: GPL v2](https://img.shields.io/badge/License-GPL%20v2-blue.svg)](https://www.gnu.org/licenses/old-licenses/gpl-2.0.en.html)
[![WooCommerce Compatible](https://img.shields.io/badge/WooCommerce-Compatible-purple.svg)]()

A lightweight, secure integration of the [Fincra Checkout](https://fincra.com/) for WooCommerce. Accept payments across Africa and internationally using Cards, Bank Transfers, and Virtual Accounts.

## âœ¨ Features
- **Easy Setup:** Plug-and-play configuration with Public/Secret keys.
- **Sandbox Mode:** Full testing support using Fincra's sandbox environment.
- **African Currency Support:** Supports NGN, GHS, KES, UGX, ZAR, and ZMW.
- **Fee Bearer Control:** Choose whether the customer or the business pays the transaction fees.

## ğŸ›  Installation

1. **Download:** Click "Code" > "Download ZIP" in this repository.
2. **Upload:** Go to your WordPress Admin > Plugins > Add New > Upload Plugin.
3. **Activate:** Once uploaded, click Activate.
4. **Configure:** Navigate to **WooCommerce > Settings > Payments > Fincra** to enter your API credentials.

## âš™ï¸ Configuration Requirements
You must have a Fincra Merchant account to use this plugin.
- Get your keys from the [Fincra Dashboard](https://dashboard.fincra.com/).
- **Business ID:** Required for Sandbox transactions.
- **Secret & Public Keys:** Ensure you use the correct keys for the selected mode (Sandbox vs. Production).
